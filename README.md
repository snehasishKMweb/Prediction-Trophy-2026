# 🏆 Prediction Trophy 2026

A modern FIFA World Cup prediction dashboard built using HTML, CSS and JavaScript.

## Features

- 📊 Live Tournament Dashboard
- 🏆 Current Leader
- ⚽ Next Match Predictions
- 📈 Tournament Statistics
- 📰 Recent Activity
- 🥇 Leaderboard
- 📖 Match History
- 🍛 Biryani Challenge
- 📱 Responsive Design

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- JSON
